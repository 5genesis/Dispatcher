# Flask Login Example with token authentication

## Requirements.txt
    Flask
    Flask-SQLAlchemy
    requests
    jwcrypto


## Setup Repo
```
virtualenv -p python3 /.
source bin/activate
git clone https://github.com/tolgahanuzun/Flask-Login-Example
cd Flask-Login-Example
pip install -r requirements.txt
python app.py
```
